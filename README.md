# erollcall-library

## Install

```bash
npm install --save erollcall-library
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'erollcall-library'
import 'erollcall-library/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
